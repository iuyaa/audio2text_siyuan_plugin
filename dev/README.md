# audio2text_siyuan_plugin

SiYuan plugin to **transcribe audio blocks into text** using OpenAI.

## Setup (OpenAI API key)

1. In SiYuan, open **audio2text → Plugin Settings**
2. Set **OpenAI API Key**
3. Save

## Use (Transcribe)

- In a document, select/right-click an **audio** block, then choose **Transcribe Audio**
- The plugin will insert the transcription as text **after the audio block**

## Notes

- Your audio is sent to OpenAI for transcription.
- Courtesy of Cursor and GPT-5.2
- Created using template from "https://github.com/frostime/plugin-sample-vite"

## 中文说明

audio2text 是一个思源（SiYuan）插件，用于使用 OpenAI 将**音频块转录为文字**。

### 配置（OpenAI API Key）

1. 在思源中打开 **audio2text → Plugin Settings**
2. 设置 **OpenAI API Key**
3. 保存

### 使用（转录）

- 在文档中选中/右键一个**音频**块，选择 **Transcribe Audio**
- 插件会把转录结果作为文本**插入到音频块之后**

### 说明

- 音频会发送到 OpenAI 用于转录。
- 由 Cursor 和 GPT-5.2 提供
- 使用以下模板创建："https://github.com/frostime/plugin-sample-vite"
